# AWS CI/CD Pipeline

## Overview
This project demonstrates an end-to-end CI/CD pipeline using AWS services. It automates the process of building, testing, and deploying applications from GitHub to Amazon EC2 using AWS CodePipeline, CodeBuild, and CodeDeploy.

- Amazon CloudWatch

## Project Workflow
1. Create a GitHub repository.
2. Launch an EC2 instance.
3. Configure IAM roles.
4. Create a CodeBuild project.
5. Configure CodeDeploy.
6. Create a CodePipeline.
7. Connect GitHub.
8. Push code to GitHub.
9. Monitor deployment using CloudWatch.
